# Scoop Bucket for Spore CLI

This is the official [Scoop](https://scoop.sh) bucket for the Spore CLI on Windows.

## Installation

```powershell
scoop bucket add sporeintel https://github.com/sporeintel/scoop-bucket
scoop install spore
```

## Updates

```powershell
scoop update spore
```

The manifest is automatically updated when new Spore CLI releases are published.

## About Spore

Spore is a distributed AI inference network. Learn more at [sporeintel.com](https://sporeintel.com).
